# 股票技術分析評分 (SCTR)

## 評分點樣計？

### 核心邏輯：技術面相對排名

系統會用 Yahoo Finance 攞最近 1 年每日價格資料，計出每隻股票嘅技術指標，再同今次股票池入面其他股票比較，計出 0-100 分嘅百分位排名。

流程：

1. 將 ticker 標準化
2. 下載 1 年日線價格同成交量
3. 用收市價計 EMA、ROC、PPO、RSI
4. 每個技術指標都同成個股票池做百分位排名
5. 每個時間維度用兩個指標分數取平均
6. 按長中短線權重加總成 `Technical Score`
7. 最後按 `Technical Score` 由高至低排序

如果某隻股票少過 201 個有效收市價，系統會直接跳過，因為計唔到可靠嘅 EMA200。

---

## 三大時間維度

| 維度                   | 睇緊咩             | 用咩指標                   | 權重    |
| ---------------------- | ------------------ | -------------------------- | ------- |
| **長線（Long Term）**  | 約半年到一年趨勢   | `EMA200Distance`、`ROC125` | **60%** |
| **中線（Mid Term）**   | 約一個月到數月趨勢 | `EMA50Distance`、`ROC20`   | **30%** |
| **短線（Short Term）** | 近日動能變化       | `PPO Slope3`、`RSI14`      | **10%** |

長線佔比最高，因為呢個分數想偏向 SCTR 式相對強弱：先睇大方向，再用中短線確認動能。

---

## 指標點計？

| 指標             | 公式 / 計法                         | 解讀                                                       |
| ---------------- | ----------------------------------- | ---------------------------------------------------------- |
| `EMA200Distance` | `(最新收市價 / EMA200) - 1`         | 價格離 200 日 EMA 有幾遠，越高代表長線越強                 |
| `ROC125`         | `(最新收市價 / 126 日前收市價) - 1` | 約 125 個交易日嘅升跌幅，越高代表半年動能越強              |
| `EMA50Distance`  | `(最新收市價 / EMA50) - 1`          | 價格離 50 日 EMA 有幾遠，越高代表中線越強                  |
| `ROC20`          | `(最新收市價 / 21 日前收市價) - 1`  | 約 20 個交易日嘅升跌幅，越高代表近月動能越強               |
| `PPO Slope3`     | `PPO.diff(3) / 3`                   | PPO 近 3 日平均變化，越高代表短線動能轉強                  |
| `RSI14`          | 14 日 RSI，使用 EMA smoothing       | 0-100 嘅相對強弱指標；呢度係排名用，唔係直接當超買超賣訊號 |

所有技術指標都係「越大越好」去排名。

---

## 時間維度分數

每個維度都係兩個指標 percentile score 嘅平均值：

```text
Long Term Score = (EMA200Distance Score + ROC125 Score) / 2

Mid Term Score = (EMA50Distance Score + ROC20 Score) / 2

Short Term Score = (PPO Slope3 Score + RSI14 Score) / 2
```

Percentile score 係相對排名：

- 股票池入面最高嗰批會接近 100
- 最低嗰批會接近 0
- 如果某個指標得一隻股票有有效數值，嗰隻會係 100
- 缺失數值會計 0

---

## Technical Score 點嚟？

`Technical Score` 係三個時間維度分數加權：

```text
Technical Score =
  (Long Term Score * 0.60)
+ (Mid Term Score * 0.30)
+ (Short Term Score * 0.10)
```

例子：

> 如果某隻股票 `Long Term Score = 90`、`Mid Term Score = 70`、`Short Term Score = 50`，咁 `Technical Score = 90 * 0.60 + 70 * 0.30 + 50 * 0.10 = 80`。

## 輸出欄位

技術面 screener 會輸出以下資料：

| 欄位                   | 意思                           |
| ---------------------- | ------------------------------ |
| `Ticker`               | 股票代號                       |
| `Quote Price`          | 最新收市價                     |
| `Quote Change`         | 最新收市價減上一個交易日收市價 |
| `Quote Change Percent` | 最新一日升跌百分比             |
| `Quote Volume`         | 最新成交量                     |
| `Long Term Score`      | 長線技術分                     |
| `Mid Term Score`       | 中線技術分                     |
| `Short Term Score`     | 短線技術分                     |
| `Technical Score`      | 技術面總分                     |
| `EMA200Distance`       | 價格相對 EMA200 偏離           |
| `ROC125`               | 125 日變動率                   |
| `EMA50Distance`        | 價格相對 EMA50 偏離            |
| `ROC20`                | 20 日變動率                    |
| `PPO Slope3`           | PPO 3 日斜率                   |
| `RSI14`                | 14 日 RSI                      |

---

## 同 Total Score 嘅關係

完整 screener 會再將技術面分數同基本面分數合成 `Total Score`：

```text
Total Score = (Fundamental Score * 0.60) + (Technical Score * 0.40)
```

即係 `Technical Score` 係總分其中 40%。如果技術面資料下載失敗、資料不足，或者計唔到技術分，完整資料仍然可能保留基本面欄位，但 `Total Score` 會係空值。

---

## 分數點解讀？

| 分數範圍   | 意思                                             |
| ---------- | ------------------------------------------------ |
| **90-100** | 技術面喺股票池入面最強，長中短線通常都有明顯優勢 |
| **70-89**  | 技術面偏強，趨勢或動能值得留意                   |
| **50-69**  | 中等，可能係整固，或者長短線有分歧               |
| **30-49**  | 偏弱，走勢唔突出                                 |
| **0-29**   | 技術面相對落後，或者關鍵價格資料缺失             |

**注意：** 呢個分數係相對排名。100 分只代表佢喺今次股票池入面技術面排得最高，唔代表一定會繼續升。

---

## 使用注意事項

1. **股票池決定排名**：同一隻股票放入唔同 ticker list，分數可以差好遠。
2. **價格資料會影響結果**：資料由 Yahoo Finance 下載，遇到停牌、缺資料、ticker 格式問題，可能會冇技術分。
3. **長線分數滯後**：EMA200 同 ROC125 反映較長週期，分數高可能代表已經升咗一大段。
4. **RSI 高唔等於買入**：呢度 RSI 只係用嚟做相對排名，高 RSI 亦可能代表短線過熱。
5. **唔係買賣訊號**：高分只係話技術面相對強，入場仍然要配合基本面、估值、風險同倉位。
